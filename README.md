# ComponentBook

A SwiftUI catalog of reusable iOS components — a "book" of building blocks for browsing, previewing, and copying components into your own apps.

## Requirements

- Xcode 15+
- iOS 17+
- Swift 5.9+

## Getting started

```sh
git clone https://github.com/iOSGaurav/component-book.git
cd component-book
open ComponentBook.xcodeproj
```

Select the **ComponentBook** scheme and run on a simulator or device (⌘R).

## Project structure

```
ComponentBook.xcodeproj/   # Xcode project
ComponentBook/             # App sources
  ├── ComponentBookApp.swift
  ├── ContentView.swift
  └── Assets.xcassets/
ComponentBookTests/        # Unit tests
ComponentBookUITests/      # UI tests
```
